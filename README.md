# THAT ACTIVE MOBB Shorts Creator

A fast, lightweight YouTube Shorts generator built for **THAT ACTIVE MOBB KREW**.  
This tool creates high‑energy GTA Online Shorts ideas, hooks, scripts, captions, hashtags, and on‑screen text optimized for the YouTube Shorts format.  

Designed for quick idea generation, consistent branding, and rapid short‑form content production for the official channel **@ThatActiveMobb**.

## Features
- Generate complete Shorts packs (hook, script, caption, hashtags)
- Crew‑branded output for THAT ACTIVE MOBB KREW
- Simple, fast, browser‑based workflow
- React + Vite project structure
- Easy to expand with AI APIs or additional components

## Project Structure
- `/src` — main application code  
- `/components` — UI and generator logic  
- `/ai` — behavior specification for the Shorts generator
